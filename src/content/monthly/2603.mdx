### 有室友在的时期

春节前后共两个月，分别有两位新来深圳的深漂朋友过来我这暂住。刚好是清明节假期第一天的今天，送走了室友 B，又回到了独自一人居住的状态。希望他们两个工作都顺利。

真是截然不同的两个人。

**室友 A**：小我七岁的一位孩子，因为未公开信息，来到深圳工作。期间一起玩了好多游戏，爬了一次山，看了一部动画、一部电影。还请我吃了一顿寿司郎。最后是年后自己找了个地方租出去了。

ISFP，该说是资讯达人和冲浪高手么？我很多时候都是将他当作更优一点的人形 GPT 来询问信息的（工具人？不是不是）。

**室友 B**：我的一位做外贸的高中同学，因公司内部转岗，升迁到深圳。期间一起去看了两部电影，爬了一次山，吃了好多顿饭。还有请我吃了一顿寿司郎（人均 180 那种）。安排上公司宿舍就搬过去了。

ENTJ，传说中的卷王，跟我同年出来工作，现在都接近有车有房了，非常恐怖。各种意义上都非常有行动力，来深圳前用了 3 个月时间减了 30 斤，体脂到了 17%。反观我都下了什么 [豪言壮志](https://niracler.com/health/) 了，现在腹部还是三节轮胎。

嗯，四舍五入，一不小心就变成每个周末都有活动，体重都恢复回去了。预计之后很长一段时间都还是维持一个人住的状态吧，暂时不再展开「收留业务」了，下次就自己解决吧～（绝对没有不耐烦的意思。只是你知道的， I 人就是要有独自的时间和空间）

### 《钢之炼金术师 FA》

<MusicPlayer id="668479" server="netease" />

跟室友 A 每天一集，陆陆续续看了一个月总算看完了。春节回家前看到大概一半，春节期间、春节过后，就是各顾各地异步看完了。要是早个十几年看，估计就刻进 DNA 了吧。自己一个人的话，五十多集大概率不敢开始看。

说起来，大概半年前跟他聊天，他说「最近看到一些漫展的视频，看见有个 Cosplay 的角色挺感兴趣的，回来一查，是一部叫《钢之炼金术师 FA》的动画」。听见这段话，我已经知道我老登得不行了～

非常王道传统的作品，音乐、人物、剧情节奏、作画都是 MAX。小时候看追的是爱德华，现在重看却更共情大人组的剧情了。上校真的是英雄，成为了社畜，看清了生活的真相后，还能有那勇气去对抗各种不公。

PS. 小时候很憧憬「画个五芒星就可以召唤土墙」的技能。太帅了，然后我也学着画，只可惜在梦中才能用上。

### 新的电脑 - MacBook Air (M5 32+512, 2026)

去年 12 月开始，我将 Claude Code 的订阅改成了 MAX 5X（5 倍用量），开发方式完全变了——写了一堆自动化 skill（[niracler/skill](https://github.com/niracler/skill)），搞了类 monorepo 的多仓库管理（[niracler/dev-config-template](https://github.com/niracler/dev-config-template)），之后大概会开几篇博客分享。

结果就是，当我尝试着开五个 workspace 的时候，我开发用的电脑（M4 mini 16g）爆内存了，偶尔会导致连打字都会卡顿。

![Apple 的 16g 内存原来是可以用到 40g 的吗（bushi）](https://image.niracler.com/2026/04/7cd1284d815ea3100d3d7893a5177202.png)

卸了 30 个插件勉强续命，但心里已经种下了「该换了」的种子。其实更早在 2024 年刚入职的时候就隐约有预感，当时日记里写的是「早知道在入职前就应该跟老大扯一下配置更高一点的」。一语成谶。

然后就是天时地利的事了。内存条涨价，下一代 Mac 铁定更贵；刚好国补窗口还开着，以旧换新也有 1000 的补贴。¥9999 国补价，旧的 M1 Air 以旧换新卖了 3000，四舍五入 7000 块拿下了这台 M5 Air。

到手第一件事是让 Claude 帮我备份旧电脑。rsync 跑了一轮：微信 28G、企业微信 3.2G、openclaw-backup 327M，合计 32G。光微信聊天记录就比操作系统还大，挺魔幻的。

![可汗大点兵，这是旧 mac 上 brew 装的软件](https://image.niracler.com/2026/03/ac676f2fbdcbcd9ac8ba4c18bf080db7.png)

至于那台 M4 Mini，现在降级成了 OpenClaw 启动器和 Surge 网关，偶尔远程连上去用用，算是发挥余热吧。（还在探索中，没找到更好的用途估计就归还公司了）

PS. 后续发现开 VS Code 其实也是多余的，大多数情况 iTerm2 直接开发就好，升级了个寂寞。当然，心情好了很多，而且也能有更多的可能性啦。

## 折腾 OpenClaw 的二三事

> 全世界铺天盖地都是 OpenClaw 的信息，我看了一下我的奈亚子，有一种被 NTR 的感觉。

OpenClaw 怎么说呢，比 Claude Code 多了常驻后台、定时任务、记忆检索之类的功能。（后续 Claude Code 也逐渐跟进了，所以我只能说 [OpenClaw，再见了](https://tg.niracler.com/posts/1634/)）

我从春节开始折腾，折腾了整整两个月，留下了两个仓库合计 96 个 commit 的记录。回过头来看，这段经历挺像是养了一只不太听话的电子宠物。

### 春节的兴奋

2 月 11 日，回到家的第二天就开始搭了。最早装在旧 MacBook 上，想法很美好：让它 24 小时挂着，定期帮我看日记、推送 RSS 日报、管理书签、推荐音乐。春节期间留下了 28 个 commit，RSS 日报的 skill 前前后后重写了至少 4 次，甚至还做了一套年度规划系统想塞进去。

朋友 L 跟我说「OpenClaw 让他有了很多奇思妙想」。确实是这样的，好工具能激发想象力，在用之前远远想不到还能这样玩。

### 3 月的头疼

春节一过，坑就来了。

第一个坑是把 MacBook 当服务器。盖上盖子就休眠，休眠了远程隧道就断，要修复休眠得先连上去，但连上去的前提是它醒着，一个完美的死循环。最后写了个脚本每秒尝试连接，趁它短暂醒来的一瞬间冲进去禁用休眠。教训：**MacBook 当服务器，第一件事是在本地禁用休眠再搬走。**

搞定之后就进入了密集配置期。让 AI 给我 929 篇 Obsidian 笔记打分筛选，挑出来的做成记忆索引；配了每半小时自动检查日记的心跳机制；加了 RSS 品味画像让它学习我的阅读偏好。3 月中旬把整套配置从旧电脑迁到了 Mac mini 上，顺便做了一轮 skill 大扫除，34 个 skill 里禁用了 22 个用不上的。

### OAuth 翻车

然后就是最惨烈的一天。3 月 25 日下午，Telegram bot 一直刷认证失败，RSS 日报也跑不出来了。折腾了大半天，最后搜到 GitHub issue 才恍然大悟：不是我的问题，是 OpenClaw 上游的 bug，权限配置缺了字段。

临时方案是切到按量付费的模型。算了一下账：光 RSS 日报单次就要 $0.70，加上其他定时任务每天约 $1.71，月费约 $51，比 ChatGPT Plus 订阅贵了一倍多。

### 现在还剩什么

虽然说了「再见」，但其实到现在（4 月初）后台还挂着，就是不再折腾新功能了。目前还在跑的 7 个定时任务：RSS 日报（每天推送到 Telegram 和企微）、笔记记忆索引、品味画像更新、网易云每日专辑推荐、音乐库完整性检查，还有自动更新。其中有几个最近在报错，但核心的日报和网易云推荐还在正常跑。

![搓了一个 RSS 每日懂我的精选文章，当其实对于我的效果还不如直接 HN 摘要的 supertechfans.com](https://image.niracler.com/2026/04/efa0f4afc5b2eadb5f57205d198cd79b.png)

其实也不亏啦。至少搞明白了 Agent 常驻这件事目前的上限在哪里（以及 macOS 休眠有多讨厌）。
