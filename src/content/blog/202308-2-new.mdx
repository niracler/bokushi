## 向现实这个烂游戏发起挑战吧

对，其实这篇文章是我对现实生活的挑战书。

### 现实是个烂游戏 - 逃避有用但可耻

**现实生活本身在我看来是由 60%的无聊 + 35% 的痛苦 + 5% 的快乐组成的**，喜剧可能是 70%快乐 + 20%痛苦 + 10%无聊， 而悲剧可能是 60%痛苦 + 20%快乐 + 20%无聊。有一些剧当我们无法共情它的快乐和痛苦的时候，那一部分的快乐或者痛苦其实就转变成了无聊，以至于我们会觉得这部剧不好看。在我看来，「 **几乎删除了无聊的部分** 」 这一点就是故事形式的作品最能吸引我的地方。毕竟不无聊就是意味着有趣啊。（<Spoiler>爆论，这一段是随口说的，别当真😄</Spoiler>）

**从某个时刻开始，我就发现了我是一个现实逃亡主义**。这可能是发生在我大学快毕业时找机器学习方面的工作受困的时候开始吧，发现现实生活确实比我想象中的要骨感的多。以至于我毕业找到工作并感觉到工作之累之后，我除了工作之外的学习时间基本上就停滞了。买上了各种游戏机和各种游戏，业余时间就沉迷在 ACG 的异世界里面，现在看来这就是一种对于现实生活的一种逃亡。

**就像很多剧里面的社畜**，他们会在下班之后会到酒吧喝酒。而我下班之后沉迷于看漫画其实也是一种类似于 **喝酒** 的行为。我很沉迷在异世界的故事里面，因为我觉得异世界的生活比我所处的生活有趣的多了。我的人生就是由漫画组成的，这话其实不假，大家看过的漫画我基本都看过（<Spoiler>400部应该是有的，tachiyomi 里面的订阅数是 415</Spoiler>）。应该说是除了学习以及工作之外，我的知识基本只有跟 ACG 有关的知识。

抖音、小红书、王者、lol、微博类似的软件或者竞技类游戏我基本都没有碰过，电视剧电影也很少看，也基本不追热点话题，线下生活更是少之又少。**以至于我没有什么正常人的常识，基本上是一个赛博电子人**。而且人天生杂念就多，日常开发的时候也经常会想异世界的一些事情。(<Spoiler>这一点的解决方案我还没想得很清楚，可能关键点是要多出门</Spoiler>)

### 对于地球 Online 的挑战书 - 制定目标

> 世界上只有一种真正的英雄主义，就是认清了生活的真相后还依然热爱它

**是的，我不想再做一个借酒消愁的人**。这一周看到了 xLog 里面这么多的后浪的小朋友的文章。他们提及到的工具，尤其是各种生活上的自动化工具如 n8n 、toggl track、auto bangumi 什么的。这让我意识到「 我已经在某种意义上脱节了 」，我还停留在 ifttt 的时代～～（其实从上一个月开始就一直是这样觉得了，所以我最近经常在切换使用不同的工具）不过我想我不用太急着去引进这些东西到我的工作流里面，我想搞清楚我现实生活中的需求在一个一个引进。毕竟没法一口吃成一个胖子。

**每月 Random 事项就是我的挑战书**。本来想着月末再整理的，做完写这个周记写着写着就索性给整理好了。我这次将 ACG 方面的事项给调到 7.6%， 之前可是高达 30%。**至于为什么是 Random ？** 因为我想做的事情其实有点多，而且随机的话有时候可能能给我带来一些随手写的惊喜的待办事项。下面列一下 Random 事项的一些要点: 

1. **时间控制在 60H 内**：关于一个月能有多少时间？之前玩八方旅人的时候 40 天可以挤出 80 个小时，那时候基本上就是下班后就开启这个游戏来玩了。现在估计下来 30 天可以有 60 个小时。
2. **要有明确的可衡量的目标**：其实也是那个「逼迫着自己输出点什么来证明自己已经懂了」的那个道理。
3. **是触手可及的事项**：最好是工作生活上遇到的事情以及问题，毕竟要有需求在，能够重复使用重复验证才是真的能做得深入。
4. **每天都要有一点点进度**：可以是相关知识点的链接分享或者是阅读评论，怎么简单先怎么来。我这边预计创建一个 daily 的仓库来专门存放这一个进度，然后通过 GitHub Action 或者 n8n 什么的技术同步到 Telegram 和 Twitter(X?)。

### 行动才是最关键的 - 如何落实

> 理论派成功人士，自己本身并不成功， 对，我就是热衷于写计划的那种人。

至此，关于我做事情以及成长的框架都已经定好了 [优雅的哲学](https://niracler/elegant) + [每周文章](https://niracler/202308-2) + [每月 Random 事项](https://niracler/202308-3)。之后很长一段时间应该不会再写这些事情了。现在这个阶段我还是说的太多做的太少了。接下来就是要用行动来证明并拔掉这些 flag 了。做事情比做计划要重要的多，口说无凭。应该是: 

1. **每周的周报**: 里面应该有我关于[优雅的哲学](https://niracler/elegant)  里面实践情况的内容。这个要朝着自动化的方向才行，每周自动生成数据什么的。
2. **Random 事项的报告** : 就在于 daily 以及一个月的其中一个周三的总结文章。

## 碎碎念

1. **这周开始恢复自己的 GitHub 的 commit 连击**。说的好像曾经有过？大学的时候应该是有过的。然后毕业之后，尤其是毕业设计写完之后就停了。不过这个事情的稍有改变的是，我找到了连击的理由了。每日写一两个工作上用的自动化的小脚本来处理一点事情什么的。
2. **写周记的一个比较大的问题就是，无法记录到细粒度的当天的现实生活的想法。** 很多时候当你一周结束要开始总结的时候，其实已经是忘记了一些事情了，尤其是周末处于放松的状态就无法共情周一到周六的痛苦了。（<Spoiler> 不过我本来就没有写现实生活中的事情的习惯～～ </Spoiler>）
3. **我想我的日记放到 LangChain 其实没什么用。** 因为我明白，我的一篇文章里面大多数情况只是输出了我的想法的一少部分，而且未经修饰之前大多数都是有逻辑问题或者是语无伦次的。
4. **《假面骑士极狐》是我人生第一次真的在追一部假面骑士。** 真的是，第一次有这种感觉，就是「 怎么这还没大结局啊，又又又出来新人了 」。简直是太有毒了，剧情怎么这么长。不过事后想一想，「哦，那个角色的故事还没讲到」，于是就感觉还是有点合理的。
5. **我相信我的工作是比其他的 90% 的「拧螺丝」类型的工作都要有趣的。** 怎么说呢，我们做的事情其实也是很有趣的事情。但是两年以来的常态化的 996 生活，以及一样事情即使再喜欢，在长久的朝夕相处之后会慢慢变得平淡起来的。于是就渐渐地感觉自己像是西部世界里面的被困在 loop 里面的机器人了，创业公司其实真的挺苦的。
6. **粗略算一下作为纯 shell 程序员，我竟然已经快 两年了**，加上一些杂七杂八的，我是不是算一个国内的小 shell 专家了？（<Spoiler>伪，不存在的，其实我觉得的对 shell 的理解也仅仅是沧海一粟</Spoiler>）
