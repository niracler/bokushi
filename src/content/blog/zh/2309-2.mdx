## 写作那些事

**试着用 1.5 小时的时间将这篇文章给写出来**。每周写周记都花了我四五个小时，我的问题在于不是能写的东西太少，而是想写的东西太多，而又不深入。以至于我的文章里挤满了各种乱七八糟内容，而没有一个比较好的中心思想，或没有将一些特定话题说清楚，都写的太浅了没有说到要点。素材在精不在多，我可能还没意识到这个问题。（<Spoiler>这一篇最后还是写了超过四个小时</Spoiler>）

这周周三写的是一个非常小的知识点，一个出乎直觉的 POSIX 行的概念。

- [为什么文件要以换行符结尾，以及我们应当如何保证 - niracler](https://niracler.com/newline)

不过也因为我记录了这么一下，我觉得这辈子都不会忘记这个知识点了。

我也想过一周更新两篇文章对于我这种 996 创业公司的社畜是否有点超过了，以至于我正常看书的时间都没有了。但是转念一想，**即使我每周不更新两篇文章，那些时间我可能也不会用来看书吧。** 再或者说，**我即使写的是月报，我可能也是在一个月的最后一天才用四个小时来写吧**。从这个角度来想的话，我应该还是要逼迫着自己做好这个事情的。正常周一到周六的过程中准备的素材完成度应该高一点，而不是单单只写一个提纲，这样我周日的时候才能轻松一点。


## TG频道近况

**TG 频道少有地多了几个人。** 只能自我安慰不用在意别人的想法。这个频道充斥着各种很粗糙的、不成熟的、或者很蠢的想法。本人的阅读量也很少，基本上可以理解为看猴子了。是那种后知后觉的没有洞见的人。

最近扫了一眼 [Yu 酱](https://t.me/pseudoyulife) 等邻居的频道后，我才发现原来这两年我错过了这么多事情。怎么说呢，好像有点想说「相见恨晚」，要是我早认识你们两年，我可能也不会堕落至今。我感觉我做了两年甚至是三年的山顶洞人。

## 工作记录

### 人际交往

**很难跟同龄人一起工作**。嫉妒之心会上来。是真的有压力啊，跟他们一起工作的时候，我觉得有点压抑和抑郁。最后导致大家都独自为战，实在是～～（<Spoiler>此处省略一千字</Spoiler>）

### 对阿里云吐槽

这周被阿里云的 cdn + oss 坑到了，应该说这个就不适合用作静态页面托管。作为一个兼任的运维工程师，我的工作真的是大失败了。

1. 最开始的时候是 index.html 都找不到，然后再是不能自动补全 .html 。
2.  content type 本身可能会因为不知名原因变了， 导致 gzip 不生效 。
3. 使用了 cdn 之后问题就更是麻烦了，没法很好的根据 oss 的更新触发 cdn 的更新，然后导致现在每次 deploy 都需要顺带跑个脚本触发更新 cdn 缓存。。
4. 就连阿里云 的 aliyun cli 也很坑，使用 config 的 delete 功能删除了默认配置之后，直接 aliyun cli 整个不能用了，要手动去清理配置文件。

### 什么人用命令行比较多

这个国庆，我们公司的第一个产品就会正式发布面向公众的第一版了。对，是一个出乎意料的命令行工具。所以一直以来这个问题都摆在我面面前，「**什么人用命令行比较多**」。

我初步有了一个回复，那些做的事情太过于前沿的人属于这一类。因为他们工具还没发展到有 GUI 的情况下就更新换代了。例如前几年前端领域更新迭代极快的时候，因为很多前端的工具在 IDE 还没做到适配时，就被更新迭代给撵走了。所以前端的同学用命令行的情况意外的还是挺多的。 

### 其他想法

有些问题，换了个方式，它就不存在了。例如像《神之天平》里面的选择，你前面救多少人都是没有什么意义的，因为最后你会改更前面的历史，然后让你救的那些人的故事就直接不存在了。没人会记得你救过她。

## 阅读 & 作品

### MyGo!!!!!

刚好在完结那天开始看，我都快震惊于我看 ACG 的作品的速度了，当天开坑，第二天看完。第一次知道 [BanG Dream!](https://zh.moegirl.org.cn/zh-tw/BanG_Dream!) 这个 IP，之前一直对搞音乐的作品都没什么感觉。这个由于节奏太好了，一下子连贯地看完了。上班的时候也会摸鱼来看别人在论坛里面的讨论。。

「**你以为是轻松的组乐队故事？大错特错，实际上是超大规模青春疼痛前女友文学**」。每个角色都有对应性格上的缺陷，里面有大量的回旋镖看得我确实有点乐了，很好很好。还有 op 涉嫌剧透，让我知道了最后组成乐队的是这几个人。。。这里面我喜欢楽奈这种的捉摸不定的无法预期其行动的猫系性格的少女。还有，我在看爱音的时候经常脑补 [爱酱大胜利](https://zh.moegirl.org.cn/zh-hans/%E7%88%B1%E9%85%B1%E5%A4%A7oo)呢

我第一次知道什么叫摇滚乐是在去年的《[地球上最伟大的一场演出 (豆瓣)](https://book.douban.com/subject/35503447/)》，不过真正涉及到摇滚乐队组成的剧情的作品是 《NANA》，时至 MyGO 也才知道了摇滚乐最少由三个人组成，鼓手、吉他、贝斯。我这回路是真的长。不过与《NANA》不同的是，《NANA》我是全程看的漫画，而 MyGO 我是看的动画，这演出效果确实就大大不一样。

不过我想开个专题考古一下「从什么时候开始，动画里面开始有乐队的剧情？」我知道的最早的是《凉宫春日的忧郁里面的》
