## 震撼世界的周末 - OpenAI Board 事件

上周六起来刷手机， Twitter 第一条就是 OpenAI 的 **Sam Altman** 被炒鱿鱼了 。原本计划好的去呼吸新鲜空气的假期，不曾想却被这突如其来的消息占据。然后那天早上我就全程在看 Twitter 上各种各样人的说法。整个周末大概有 三分之一的时间都在关注这个事情（<Spoiler>真是的，还我一个周末，八卦之心太强了。吃这瓜吃得真上瘾</Spoiler>）

这个事情过程中多次反转，不到最后都不知道这个事情的主导者是谁。而期间的舆论几乎是一边倒地站在 **Sam Altman** 一方。最后是以 Sam 快速通关乔布斯副本为结局。我是没有能力去评论这个事情，而且在这一周内网上已经有无数人分析了无数次这个事情了。只能说，我只是偶然成为了这场风波的见证者。最后公司里面七百多人起义反抗董事会的场景真是壮观，等后面出电视剧了🤣

才发现，在此事件之前，我对 OpenAI 其实一无所知。虽然天天在用，却从未深入了解其背后的组织机构。这已经不仅仅是「灯下黑」这种事情就能解释清楚的，更多体现在我的日常习惯——对于世间事务的不加思索。不仅仅是 OpenAI，看动画漫画的时候我也无法分辨哪个角色是哪个声优配音的，再或者说漫画里哪个分镜好，好在哪里，我一点也说不出来。游戏也是如此，别人玩了就好像深入了他的骨髓，事后可以对这个游戏的各处细节都滔滔不绝。而我玩过的游戏犹如过眼云烟，匆匆掠过，留下的只有片刻的快感，却无法沉淀为长久的记忆。「**要用软件，更要了解软件背后的公司以及人。要看作品，更要了解作品背后的作者**。」这个其实也是我一直想做到而又做不到的事情。

这个事情我才慢慢地知道什么叫做领导力，Sam 这就叫做领导力。以及才明白「**万物皆草台**」这个梗，但是我觉得草台和草台之间其实差别是非常大的。其实是相对的，视角不一样也会有很大的不同，有时候用显微镜去看各种东西，你会觉得哪哪都是漏洞。

(OpenAI 这件事情已经进到 HN 里讨论热度最高的话题第三了。前 5 里面有 3 篇都是讣告，第一是霍金去世，这事情已经比当年乔布斯去世热度还高了。)

## 关于生活的记录

这两周我的 Telegram 频道人数出现了激增，一下子在两周之内就翻倍了。因为一不小心就被拉到 [🌍 **網路上的芳鄰**](https://t.me/addlist/bSV5jwS5TOA4MDll) 列表上了。（<Spoiler>一不小心成为了最拉跨的那一个</Spoiler>）

我自问并不清楚是如何一步步搭建起与其它频道的关系网络，或许，这一切得益于初入公司时，我以开源社区的身份加入了 Tuna 群组。不过在那群里我完全插不上话，全是高端对话， 95% 的情况下其实都是完全不清楚里面在聊什么。但正是这些日常接触，让我认识到自己是一个「**什么都不知道的笨蛋**」，并且让我从 Twitter 朋友和 Telegram 邻里那里学到了许多知识。（<Spoiler>或许用“学习”一词并不准确，更像是在各方面开阔了我的视野</Spoiler>）

在目睹其他高质量频道的内容时，我不禁反思自己每日滔滔不绝地发碎碎念发，且内容质量让人堪忧。这个其实是不好的，所以我现在尽量将内容精炼一下，先在一个私有的频道整理好信息，然后再更新到我的公开频道里面。然后限制一下一周下来平均每天最多发两篇吧。也不能不说话，不然连犯蠢的机会都没有了。

### 出去走走 - 停不下来的倾诉欲

上周末，我邀约了一位同学，一同前往邻近小镇参观博物馆，体验一番文化的韵味。尽管我们只是匆匆浏览了一两个展览，但这份体验本身就是一次宁静的享受。然后晚上两个人吃了一顿很丰盛的餐，完全吃不完，哈哈。不过这走走是好的，我真巴不得每周都去外面走走。不过工作上的原因，周末也完全没有时间呢～

我也发现我太久没有跟人聊天了，在上学时期，其实我是字条狂魔来着，天天不跟别人聊天的话，总会觉得少了一点什么。应该说是「要给自己找一些存在的价值么」

已经好几年没有跟别人在线下聊天聊到口干舌燥了。

下面这是 GPT 修饰之后的版本，感觉已经不是我说的了：

> 晚间，我们一同享用了一顿丰富的晚餐，虽然分量让我们有些望而却步，我们终是不禁哑然失笑。真心希望能有更多机会，让脚步不再局限于工作的圈子，而是走进广阔的天地。不过随着工作的忙碌，如今能够享受周末的日子仿佛成了奢侈。

> 在轻松的聊天中，我发现自己竟有些口干舌燥——这种感觉，似乎已经消失了很久。曾经的我，在校园生活里，总是习惯于与人交谈，似乎每天的对话都是维系自我的方式，是我寻找自我存在感的途径。而如今沉浸在工作中，那种不断交谈的时光已变得难以寻觅。这次的线下倾谈仿佛唤醒了沉睡的自我，我开始渴望重新找回与人沟通的美好，却又不知何从下手。

> 这简单的逸闲行程让我体悟到，生活不仅仅局限于赚钱和事业，更关乎于与人的交流和内心的沟通。我们每个人都渴望被听见、被理解，在匆忙的世界中寻找自我的价值与定位。也许，这就是我要刻意去寻找和维护的“周末时光”——一个充满真情实感的空间，一个能让自我表达和倾诉欲望得到满足的场合。

## 找点事做

看见霍霍好可爱，于是将星铁下载回来了。不行，我不能玩手游，手游会让我不幸的。详情见我一年前的 [最大最黑事件](https://niracler.com/shoujo-kaisen)。感觉手游只要还是这种抽卡的模式，我这辈子都不想再接触了。停手停手，明明 P5R 都还没通关第一个鸭志田来着。

我一直以来的最大的特点就在于，经常在说，但很少去做。其实我焦虑的原因也在于此，于是这两周就开始给自己找点事情做了。（<Spoiler>这里指的是做的过程中需要动脑的事情，所以看漫画是不算的</Spoiler>）。

### AutoBangumi 自动化追番

这个事情源于 haruki 酱:

> 快来小祥の小窝一起追番吧 😇 [https://pan.sakiko.top](https://t.co/NeUJsqaTAF)

**于是花一个晚上将 AutoBangumi 搭起来了**。现在是部署在宿舍的 steam deck 上，全天候运行当成服务器来用。看动画的时候是用 iPad 在床上通过 nplayer 播放器用 sftp 协议连过来看。有了这个之后，真的可以在第一时间追番了，因为一但更新就会同步信息到我的 telegram 上～～ 至于是否要跟他们一样 post 到云服务器上，我到觉得我在宿舍之外的地方应该不会去看番吧。

看到比我小七八岁的你们在折腾「自动化追番」，我这颗老心脏仿佛被点燃了青春的火焰。勇往直前的心已经消逝多年，我发现我从工作开始就一直没有在思考了。这几年皆是如此，明明我从大一开始就全环境用 Linux ，明明大学期间帮同学装 Linux 系统的次数都快比班上同学人数都多了，但是现在工作之后折腾的次数就越来越少了。（<Spoiler>可能是累了？</Spoiler>）

**大概是从大学开始，我就渐渐地减少看动画的时间了**。高中三年虽然时间跟现在一样紧张，但那个时候每一季都扫地式地去看的。相对于看动画我现在看漫画会看得多得多。动画一集的时间是 20 多分钟，启动成本会比漫画高不少。不过，漫画也会出现看个不停的情况。现在每一季其实都只是在追那一两部动画，例如这一季就只在看芙莉莲，不过其实主要是因为我大多在漫画阶段就已经看了。也可能我眼光更高了，看不了那种一般的作品了。

### 在 Cloudflare 上搭了一个 Telegram 同步 Twitter 的机器人

也是因为上面 OpenAI 这个事情，我一不小就养成了看 Twitter 的习惯了，也一次过关注了一批高质量的推主。

**我的倾诉欲比较强，经常会有很多话想说**。一直以来，我在 Telegram 上说的话比所有其他平台加起来的都要多。一般碎碎念什么的就是贴在我的 channel 上的。可能是因为我是独生子，也可能是因为我没有什么朋友。所以写 blog 写周记一直以来是我最好的情绪发泄点。表达欲特别强，然后又很菜？

不，才不是因为在 Telegram 上发碎碎念没有人点赞😳，于是为了传播到 Twitter 上去才搭建的机器人。实际上是为了学习一下 Cloudflare Workers 的使用，顺便把 Telegram 的机器人功能也实现了。作为练手，毕竟 Cloudflare Workers 的免费额度大，而且 Telegram 机器人也不用特地去设置代理，就可以直接使用。（<Spoiler>哈哈，试图掩饰</Spoiler>）

然后昨天下午就开干了。这个事情前前后后花了不到一天时间，但做起来我觉得是我今年少有的处于心流状态的编码经历了，真的是全神贯注。在做这个事情的过程中，我极大地收到了 「GPT3.5 -> GPT4」的之间的切换的红利。

也**踩了很多坑**， twitter 的 OAuth1.1a 验证卡到我了，以及 v2 的 API 才能用于发 tweet 而 v1.1 的 API 才能用于上传图片，这也是恶心到我了。而且中间还有一段是因为我的 token 被轮换导致失效，以及签名少了个字段一直验证失败～～

## 后记

下周的话，公司的工作压力就到我身上了。可能我会消失一段时间，或者说状态会变差。但其实我已经承认了这个就是生活的真相，我唯一能做的，其实就是去热爱它。我现在天天会脑补自己处于公司舆论的最低点。虽然这一点我还是一如既往，但是焦虑并不能解决我的状况的，于是这两周的折腾其实就是对我自己之前的颓废的状态的一个简易回复吧。当然了，我要在现实中去认识更多的人和去更多的地方。不能天天宅在家里对着桌面上的休比在那傻笑。
